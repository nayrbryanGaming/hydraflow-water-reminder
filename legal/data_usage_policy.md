# Data Usage Policy

HydraFlow adheres to strict data minimization principles.

## Data We Process
- **Authentication Data**: Managed by Firebase Auth.
- **Hydration Logs**: Stored in Firestore.
- **Biometric Data**: Used locally and synced for your convenience.

## Third-Party Services
We use Firebase (Google) for backend services. No data is sold to third parties.

## Analytics
We use Firebase Analytics to improve app performance and user experience. All analytics data is anonymized.
