# HydraFlow Branding Specifications

## Logo Concept
A minimalist water droplet integrated directly into a dynamic circular progress ring. 
This symbology combines the core element of the app (hydration/water) with the behavioral element (progress rings/habit formation).

## Logo Generation Prompt
Use this prompt with Midjourney, DALL-E, or your designer:
`"minimalist water droplet combined with circular progress ring, modern health tech branding, blue aqua gradient palette, clean vector icon, mobile app icon style, flat design, white background"`

## Color Palette

| Color Name | HEX Code | Usage |
| --- | --- | --- |
| **Primary Blue** | `#2F80ED` | Primary buttons, active tabs, progress rings |
| **Secondary Aqua** | `#56CCF2` | Gradients, secondary items, highlights |
| **Accent Mint** | `#6FCF97` | Success states, achievements, completed goals |
| **Background Dark** | `#0A1628` | Dark mode backgrounds |
| **Background White** | `#F9FBFF` | Light mode backgrounds |

## Typography
**Primary Font:** `Outfit` (Google Fonts)
Usage: Clean, modern sans-serif perfectly legible for analytical dashboards and large numbers.

## Voice & Tone
- Motivating, never judgmental.
- Clean and clinical, but friendly. 
- Focuses heavily on "streaks" and "consistency".
